# sample_ble

こちらは cocoapods を使用しているので、ビルドする場合は以下をお試しください。

#### 1. ターミナルで以下のコマンドを打ってください。  
すでに`cocoapods`がインストールされている場合はこの手順をスキップしてください。  
```Shell
sudo gem install cocoapods
```
#### 2. プロジェクトにライブラリをインストールします。  
初めてプロジェクトにライブラリを導入するときは、  
```shell
pod install
```
2回目以降(ライブラリの追加・削除)は、  
```shell
pod update
```
